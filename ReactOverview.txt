useState: Example of making a state of Alert 
    -import React ,{useState} from 'react';
    -const[alert ,setAlert ]=useState()                   //This is the object now. alert is a state variable while setAlert is the method to set(change the state) of the alert .This is generic way of useState Text is the original form while if we hv to change the original form we can use SetText 
    -write a func:
        const showAlert =(message,typeOfAlert) =>{          //setAlert it takes 2 parameters the msg and what's its type i.e success/fail
			setAlert({
				msg: message,
				type: typeOfAlert
		    });
	    }
    -now to use this parameters in the component file we can use 
        {props.alert.type}
        {props.alert.msg}
    -we have to pass the showAlert func to the component files like (TextForm) as we update or use any feature it shows alerts like if uppercase was clicked then it will show a alert that it was success
